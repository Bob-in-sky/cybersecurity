# Protocol Ports

The table below summarizes the default port numbers of the protocols covered in this notebook.

| **Protocol** | **Transport Protocol** | **Default Port Number** |
| :--- | :--- | :--- |
| TELNET | <span style="color: inherit;">TCP</span> | 23  |
| <span style="color: inherit;">DNS</span> | <span style="color: inherit;">UDP</span> or <span style="color: inherit;">TCP</span> | 53  |
| <span style="color: inherit;">HTTP</span> | <span style="color: inherit;">TCP</span> | 80  |
| HTTPS | <span style="color: inherit;">TCP</span> | 443 |
| <span style="color: inherit;">FTP</span> | <span style="color: inherit;">TCP</span> | 21  |
| <span style="color: inherit;">SMTP</span> | <span style="color: inherit;">TCP</span> | 25  |
| <span style="color: inherit;">POP3</span> | <span style="color: inherit;">TCP</span> | 110 |
| <span style="color: inherit;">IMAP</span> | <span style="color: inherit;">TCP</span> | 143 |
